# Changelog

## [0.1.0-alpha] - 2026-05-17

### Added
- Field arithmetic (BN254, BLS12-381, Goldilocks)
- NTT processor with GPU butterfly kernel
- MSM processor with Pippenger's algorithm
- R1CS constraint system
- Groth16 proof system
- PLONK proof system
- STARK proof system
- FRI polynomial commitment
- Proof serialization (JSON, snarkjs format)
- CLI with prove/verify/benchmark commands
- Benchmark suite
- Configuration system
- Test suite
- Docker + ROCm support
